# 更新日志

## [1.1.0] - 2026-07-03

### 新增
- 生成过程可随时点 ⏹ 停止，已生成的部分会保留
- 推理模型（deepseek-reasoner 等）的思考过程实时显示在可折叠区域
- 代码块带语言标签和一键复制按钮
- Markdown 表格渲染
- 模型配置支持单独设置温度和最大回复 tokens
- 宽屏模式下 ⚙ 改为打开右侧设置面板，不再弹窗遮挡聊天

### 修复
- 开启上下文记忆时，最新一条消息被重复发送两遍的问题
- 上传的图片现在会真正随请求发送给视觉模型（之前只在本地显示）
- 代码块内的 `*` `**` 不再被误解析为斜体/粗体
- 多个列表之间的正文不再被错误合并进列表
- AI 回复中的链接过滤掉 `javascript:` 等危险协议
- 流式回复中断时不再永久卡住发送按钮
- 流式输出不再每个字符都重新高亮全部代码，长回复不卡顿
- 往上翻看历史时，流式输出不再强行把页面拽到底部

## [1.0.0] - 2025-05-05

### 新增
- 多模型支持，内置 8 家服务商预设（DeepSeek、通义千问、硅基流动、GLM、Kimi、百川、零一万物、MiniMax）
- 实时流式输出，AI 回复逐字显示并支持 Markdown 渲染
- 多轮上下文记忆，可自定义携带历史消息条数
- 亮色 / 深色主题，4 种预设主题色 + 自定义取色器
- 紧凑模式（手机）和宽屏模式（桌面）双布局
- 多对话管理，支持标签分类（创建、切换、删除、清空）
- 对话导出为 Markdown，模型配置导出 / 导入为 JSON
- 支持图片上传给多模态模型
- 一键重新生成 AI 回复
- 自动获取模型列表
- 连接测试工具
- 安全防护：内网访问拦截、IP 频率限制、可选 AUTH_KEY 认证
- Docker 部署支持
